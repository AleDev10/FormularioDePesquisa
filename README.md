# 📋 Formulário de Pesquisa

Sistema web para coleta e gerenciamento de informações através de formulários personalizados.

## 🚀 Sobre o Projeto

O Formulário de Pesquisa é uma aplicação web desenvolvida para permitir que usuários preencham formulários online de forma simples e organizada.

Os dados podem ser processados pelo servidor e utilizados para análises, relatórios ou armazenamento em banco de dados.

## ✨ Funcionalidades

- Cadastro de informações através de formulários
- Validação de campos
- Interface responsiva
- Processamento de dados no servidor
- Suporte para hospedagem na Vercel
- Geração de relatórios (quando configurado)

## ⚙️ Instalação

### 1. Clonar o repositório

```bash
git clone https://github.com/AleDev10/FormularioDePesquisa.git
```

### 2. Entrar na pasta

```bash
cd FormularioDePesquisa
```

### 3. Instalar dependências

```bash
npm install
```

### 4. Executar o projeto

```bash
npm start
```

ou

```bash
node server.js
```

## 🌐 Acesso
